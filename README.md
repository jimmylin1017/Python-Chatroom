# Network Applications Programming Final Project
### Python-Chatroom with File transmission
#### File
* client(.py)
	* client main program
* myclient(.py)
	* client class implementation
* server(.py)
	* server main program
* myserver(.py)
	* server class implementation
* mininetENV(.py)
	* mininet network environment
#### Feature
1. unicast chat
2. broadcast chat
3. upload file
4. download file
#### Usage command
* unicast chat
```
 /chat <user> <chat_conent>
```
* broadcast chat
```
 <chat_conent>
```
* list file on server
```
 /ls
```
* upload file
```
 /up <file_name>
```
* download file on server
``` 
 /down <file_name>
```
* logout and terminal
```
 /logout
```
